# Contributor Covenant Code of Conduct

This project adopts the **[Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** in full. The short version:

- Be respectful and inclusive.
- Disagree about ideas; never attack people.
- Assume good faith. Apologise when you got it wrong.

## Scope

This Code applies to all project spaces — GitHub issues, PRs, discussions, code review, and any associated chat or events.

## Reporting

Violations may be reported privately by opening a GitHub security advisory or by emailing the maintainers (address in `pyproject.toml`). Reports are confidential and acknowledged within 72 hours.

## Enforcement

Maintainers may remove, edit, or reject contributions and ban contributors for behaviour they deem inappropriate, threatening, or harmful — proportionally to the severity of the incident as laid out in the Contributor Covenant 2.1 *Enforcement Guidelines*.
